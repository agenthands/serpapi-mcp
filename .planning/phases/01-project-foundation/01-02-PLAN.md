<task type="auto">
  <name>Task 2: Create goreleaser configuration for multi-platform builds</name>
  <files>.goreleaser.yml</files>
  <read_first>
  - .planning/phases/01-project-foundation/01-CONTEXT.md (locked decisions D-10 through D-14)
  - go.mod (verify module path for build config)
  </read_first>
  <action>
  1. Create `.goreleaser.yml` per D-10 through D-14:
     ```yaml
     build:
       - id: serpapi-mcp
         main: ./cmd/serpapi-mcp
         binary: serpapi-mcp
         env:
           - CGO_ENABLED=0
         goos:
           - linux
           - darwin
           - windows
         goarch:
           - amd64
           - arm64
         ignore:
           - goos: windows
             goarch: arm64
         ldflags:
           - -s -w -X main.version={{.Version}} -X main.commit={{.Commit}} -X main.date={{.Date}}

     archives:
       - format: tar.gz
         name_template: >-
           serpapi-mcp-
           {{- .Version }}-
           {{- .Os }}-
           {{- .Arch }}
         format_overrides:
           - goos: windows
             format: zip

     checksum:
       name_template: "checksums.txt"
       algorithm: sha256

     snapshot:
       version_template: "{{ .Version }}-next"

     release:
       mode: replaces
     ```
     Per D-10: naming convention `serpapi-mcp-{version}-{os}-{arch}`.
     Per D-11: .tar.gz for Linux/macOS, .zip for Windows.
     Per D-12: Tag-triggered releases; CI will use `--snapshot` for build verification.
     Per D-13: SHA256 checksums file included.
     Per D-14: Five target platforms. The `ignore` field excludes windows/arm64 (not a target), giving:
       - linux/amd64, linux/arm64, darwin/amd64, darwin/arm64, windows/amd64 = 5 platforms.

  2. Add version variables to `cmd/serpapi-mcp/main.go` to support goreleaser ldflags:
     ```go
     package main

     import (
         "fmt"
         "os"

     var (
         version = "dev"
     ```

     These variables are set by goreleaser via ldflags at build time.

     ```bash
     goreleaser release --snapshot --clean 2>/dev/null || echo "goreleaser not installed locally — CI will validate"
     ```
