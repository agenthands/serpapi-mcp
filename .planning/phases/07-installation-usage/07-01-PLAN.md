**Structure:**

1. **Title and intro** — "Installation" heading, brief description: "Install serpapi-mcp on Linux, macOS, or Windows."

2. **Prerequisites** (per D-05)
   - Go 1.25+ required for `go install` and `go build` methods
   - SerpApi API key from https://serpapi.com/dashboard
   - Note: Binary download method requires NO Go installation

3. **Binary download** (INST-01, per D-04)
   - Link to GitHub Releases: https://github.com/agenthands/serpapi-mcp/releases
   - Platform-specific download commands for all 5 targets:
     - linux/amd64: `curl -L` + tar extraction command, `chmod +x`
     - linux/arm64: same pattern with arm64 archive
     - darwin/amd64 (macOS Intel): same with darwin-amd64 archive
     - darwin/arm64 (macOS Apple Silicon): same with darwin-arm64 archive
     - windows/amd64: PowerShell download, .zip extraction, note that executable is `serpapi-mcp.exe`
   - Version verification: `./serpapi-mcp --version`
   - Upgrade instructions: download new version, replace binary

4. **go install** (INST-02)
   - Command: `go install github.com/agenthands/serpapi-mcp@latest`
   - Binary installed to `$GOPATH/bin` (or `$HOME/go/bin`)
   - Add `$GOPATH/bin` to PATH if not already
   - Version check: `serpapi-mcp --version`
   - Upgrade: re-run `go install` command with new version tag
   - Note: requires Go 1.25+

5. **Build from source** (INST-03)
   - Clone: `git clone https://github.com/agenthands/serpapi-mcp.git && cd serpapi-mcp`
   - **go build method:**
     - `go build -o serpapi-mcp ./cmd/serpapi-mcp`
     - Version will show "dev" unless ldflags are set
     - For versioned build: `go build -ldflags "-s -w -X main.version=X.Y.Z -X main.commit=$(git rev-parse --short HEAD) -X main.date=$(date -u +%Y-%m-%dT%H:%M:%SZ)" -o serpapi-mcp ./cmd/serpapi-mcp`
   - **goreleaser method:**
     - `go install github.com/goreleaser/goreleaser/v2@latest`
     - `goreleaser build --snapshot --clean` (local build, no release)
     - `goreleaser release --clean` (full cross-platform release, needs Git tag)
   - Verification: `./serpapi-mcp --version`
   - Upgrade: `git pull && go build` or `git pull && goreleaser build`

6. **Docker (note as out of scope)**
   - Brief note: "Docker deployment is not supported. The server ships as static binaries — see binary download or go install above."

7. **Next steps**
   - Link to USAGE.md for configuration and running
   - Link to README.md for quick start
